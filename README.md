# FIA Farm API

Este é um projeto básico utilizando **TypeScript** e **Express** para criar uma API.

## Estrutura do Projeto

```
fia-farm-api/
├── public/
│   ├── index.html
│   └── stylesheets/
├── src/
│   ├── app.ts
│   ├── interfaces/
│   │   └── http/
│   │       ├── routes/
│   │       │   ├── index.ts
│   │       │   └── users.ts
│   │       └── ErrorHandler.ts
├── README.md
├── package.json
└── tsconfig.json
```

## Inicie o servidor:
```bash
npm run dev
```
